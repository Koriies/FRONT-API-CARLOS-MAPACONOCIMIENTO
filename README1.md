# 🧠 Mapa de Conocimiento - CRUD

Sistema CRUD desarrollado con:

- 🔹 API REST en .NET (C#)
- 🔹 PostgreSQL (pgAdmin)
- 🔹 Frontend en Flask (Python)

## 🚀 Funcionalidades

- CRUD completo de entidades
- Relación muchos a muchos (Proyecto - Productos)
- Procedimientos almacenados
- Arquitectura desacoplada

## 🛠️ Tecnologías

- .NET 9
- PostgreSQL
- Flask
- Bootstrap

## ▶️ Ejecución

### Backend (.NET)
dotnet run

### Frontend (Flask)
python app.py
